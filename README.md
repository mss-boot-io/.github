# mss-boot-io organization defaults

This repository stores the public GitHub profile, default community health files,
and reusable automation workflows for the `mss-boot-io` organization.

## Purpose

`mss-boot-io` is a governance-first, agent-readable Go microservice admin
framework organization. The core projects focus on production maintenance,
clear release environments, security scanning, and AI-assisted contribution
workflows.

## Core repositories

- `mss-boot`: base Go framework.
- `mss-boot-admin`: admin backend and API service.
- `mss-boot-admin-antd`: admin web console.
- `mss-boot-docs`: documentation, organization memory, and public operations
  records.

## Defaults stored here

- Organization profile: `profile/README.md`
- Community health defaults: `CODE_OF_CONDUCT.md`, `CONTRIBUTING.md`,
  `SECURITY.md`, `SUPPORT.md`
- Default issue and PR templates under `.github/`
- Reusable GitHub Actions workflows under `.github/workflows/`

Repository-specific files override these defaults when a project needs stricter
rules.
