# Support

## Where to ask

- Usage questions: GitHub Discussions when enabled, otherwise repository issues
  with the `type/question` label.
- Bug reports: use the bug report issue template.
- Feature proposals: use the feature request issue template.
- Security reports: follow `SECURITY.md`; do not open public issues.

## Response model

`mss-boot-io` is maintained by a personal/open source maintainer model. AI
automation helps triage and summarize issues, but maintainers may respond in
batches based on release priorities.

Please include enough context for another contributor to reproduce or verify the
request without private access.
